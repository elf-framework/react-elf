# Alert

The `alert` banner displays emergency and high signal messages, such as system warnings. This is to pay attention and urge the user to take action.

## Usage

<Unstyled>
  <MainView />
</Unstyled>

<Canvas of={AlertStories.Default} sourceState="shown" />

## Options

### Text

Text is required for all alert banners. The message should be concise
and, if applicable, describe the next step that a user can take.

<Unstyled>
  <TextView />
</Unstyled>

<Canvas of={AlertStories.Text} sourceState="shown" />

### Variant

An alert banner always has a semantic meaning and uses semantic
colors. Only gray (neutral), blue (informative), and red (negative)
are available as options.

  <VariantView />

An alert banner always has a semantic meaning and uses semantic

### Text overflow

When the text is too long for the available horizontal space, it wraps
to form another line. In actionable alert banners, the button moves
below the text prior to text wrapping.
